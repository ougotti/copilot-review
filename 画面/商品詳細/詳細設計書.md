---

## 1. 処理フロー

```
[ページ表示]
    │
    ├─ URLパラメータから product_id 取得（型チェック: 整数。不正な場合 400 / 存在しない場合 404）
    │
    ├─ products テーブルから商品情報取得
    │       is_published = 1 でなければ 404 レスポンス
    │       DBエラー → 500エラー画面（SCR-ERR）・ERRORログ出力
    │
    ├─ product_images テーブルから画像一覧取得（sort_order昇順）
    │
    ├─ 同一カテゴリの関連商品を最大4件取得（自身を除く）
    │
    ├─ HTTP キャッシュヘッダー付与（Cache-Control: public, max-age=60）
    │
    └─ レスポンス生成・描画

[カートへ追加]
    │
    ├─ 認証チェック: セッションCookieが存在するか確認（ゲストもセッション使用）
    │
    ├─ CSRFトークン検証
    │
    ├─ リクエスト: product_id（integer, required）, quantity（integer, required, min:1, max:9）
    │
    ├─ クライアントサイド: 「カートに追加」ボタンを押下後にロック（二重送信防止）
    │
    ├─ [トランザクション開始]
    │       ├─ 在庫チェック: SELECT FOR UPDATE（行ロック取得）
    │       │       stock >= quantity でなければ rollback → 409 エラー
    │       │
    │       └─ cart_items へ UPSERT
    │               既存カートアイテムがあれば数量を加算
    │               加算後の数量が在庫数を超える場合は在庫数を上限とする
    ├─ [トランザクションコミット]
    │
    ├─ DBエラー → rollback → 500エラー・ERRORログ出力
    │
    └─ カート画面へリダイレクト
```

## 2. API定義

### GET /api/products/{id}

**認証**: 不要（公開API）

**パスパラメータ**: `id` – integer, 必須

**レスポンス (200 OK)**

```json
{
  "id": 1,
  "name": "サンプル商品A",
  "price": 1980,
  "price_with_tax": 2178,
  "description": "<p>商品説明HTML</p>",
  "stock": 5,
  "is_available": true,
  "images": [
    { "url": "/images/1_main.jpg", "sort_order": 1 },
    { "url": "/images/1_sub1.jpg", "sort_order": 2 }
  ],
  "related_products": [
    { "id": 2, "name": "関連商品B", "price": 2500, "thumbnail_url": "/images/2.jpg" }
  ]
}
```

**エラーレスポンス**

| ステータス | エラーコード | 説明 |
|-----------|-------------|------|
| 404 | `NOT_FOUND` | 商品が存在しない / 非公開 |
| 500 | `INTERNAL_SERVER_ERROR` | DBエラー |

```json
// 404
{ "error": "NOT_FOUND", "message": "商品が見つかりません" }
// 500
{ "error": "INTERNAL_SERVER_ERROR", "message": "システムエラーが発生しました。しばらく後に再度お試しください" }
```

### POST /api/cart/items

**認証**: セッションCookie必須（ゲストセッション含む）

**CSRFトークン**: リクエストヘッダーに `X-CSRF-Token` を含めること

**リクエストボディ**

| フィールド | 型 | 必須 | 説明 |
|-----------|-----|------|------|
| product_id | integer | ○ | 商品ID |
| quantity | integer | ○ | 数量（最小:1、最大:9） |

```json
{
  "product_id": 1,
  "quantity": 2
}
```

**レスポンス (201 Created)**

```json
{
  "cart_item_id": 42,
  "message": "カートに追加しました"
}
```

**エラーレスポンス**

| ステータス | エラーコード | 説明 |
|-----------|-------------|------|
| 400 | `VALIDATION_ERROR` | quantity範囲外など |
| 409 | `OUT_OF_STOCK` | 在庫不足 |
| 500 | `INTERNAL_SERVER_ERROR` | DBエラー |

```json
// 400
{ "error": "VALIDATION_ERROR", "message": "数量は1〜9の範囲で指定してください" }
// 409
{ "error": "OUT_OF_STOCK", "message": "在庫が不足しています" }
// 500
{ "error": "INTERNAL_SERVER_ERROR", "message": "システムエラーが発生しました。しばらく後に再度お試しください" }
```

## 3. DB アクセス詳細

### テーブル: `product_images`

| カラム | 型 | 説明 |
|--------|----|------|
| id | INT PK | |
| product_id | INT FK | |
| image_url | VARCHAR(500) | |
| sort_order | INT | 表示順 |

### テーブル: `products`（参照カラム追記）

| カラム | 型 | 説明 |
|--------|----|------|
| id | INT PK | 商品ID |
| name | VARCHAR(200) | 商品名 |
| price | INT | 税抜価格 |
| description | TEXT | 商品説明（HTML。XSS対策はフロントエンドでDOMPurifyにより無害化） |
| stock | INT | 在庫数 |
| is_published | TINYINT(1) | 公開フラグ |
| category_id | INT FK | カテゴリID |

### テーブル: `cart_items`

| カラム | 型 | 説明 |
|--------|----|------|
| id | INT PK | |
| session_id | VARCHAR(128) | セッションID |
| product_id | INT FK | |
| quantity | INT | |
| created_at | DATETIME | |
| updated_at | DATETIME | |

**UNIQUE制約**: `(session_id, product_id)` にユニーク制約を設け、UPSERT で重複追加を数量加算に変換する。

### カート追加 UPSERT SQL

```sql
INSERT INTO cart_items (session_id, product_id, quantity, created_at, updated_at)
VALUES (:session_id, :product_id, :quantity, NOW(), NOW())
ON DUPLICATE KEY UPDATE
  quantity = quantity + :quantity,
  updated_at = NOW();
```

## 4. 税込価格計算

- 消費税率: 10%（定数管理、将来の税率変更に対応）
- 計算式: `price_with_tax = FLOOR(price * 1.10)`

## 5. エラーハンドリング

| エラー | 対応 | ログレベル | ログ内容 |
|--------|------|-----------|----------|
| 商品が存在しない / 非公開 | 404ページ表示 | INFO | リクエストURL・product_id |
| 在庫不足 | 409レスポンス「在庫が不足しています」 | INFO | product_id・要求数・在庫数 |
| POST バリデーションエラー | 400レスポンス | INFO | バリデーションエラー詳細 |
| カート追加失敗（DBエラー） | 500エラー画面（SCR-ERR）・エラーメッセージ表示 | ERROR | リクエストID・セッションID・SQLエラーコード |
| DBタイムアウト（3秒） | 500エラー画面表示 | ERROR | タイムアウト発生箇所 |
| CSRFトークン不一致 | 403レスポンス | WARN | リクエストIP・セッションID |

**エラーレスポンスのセキュリティ**: 500エラー画面にスタックトレース・DB内部情報を含めない。ERRORログにのみ記録する。

**description のXSS対策**: APIレスポンス中の `description`（HTML）はフロントエンドにてDOMPurifyで無害化してからDOMに挿入する。

**タイムアウト設定**: DBクエリタイムアウト3秒（接続タイムアウト1秒）。
