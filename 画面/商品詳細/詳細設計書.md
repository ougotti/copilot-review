## 1. 処理フロー

```
[ページ表示]
    │
    ├─ URLパラメータから product_id 取得
    │
    ├─ products テーブルから商品情報取得
    │       is_published = 1 でなければ 404 レスポンス
    │
    ├─ product_images テーブルから画像一覧取得（sort_order昇順）
    │
    ├─ 同一カテゴリの関連商品を最大4件取得（自身を除く）
    │
    └─ レスポンス生成・描画

[カートへ追加]
    │
    ├─ リクエスト: product_id, quantity
    │
    ├─ 在庫チェック: stock >= quantity
    │       不足時 → エラーメッセージ表示
    │
    ├─ sessions / cart_items テーブルへUPSERT
    │       既存カートアイテムがあれば数量を加算
    │
    └─ カート画面へリダイレクト
```

## 2. API定義

### GET /api/products/{id}

**レスポンス (200 OK)**

```json
{
  "id": 1,
  "name": "サンプル商品A",
  "price": 1980,
  "price_with_tax": 2178,
  "description": "<p>商品説明HTML</p>",
  "stock": 5,
  "is_available": true,
  "images": [
    { "url": "/images/1_main.jpg", "sort_order": 1 },
    { "url": "/images/1_sub1.jpg", "sort_order": 2 }
  ],
  "related_products": [
    { "id": 2, "name": "関連商品B", "price": 2500, "thumbnail_url": "/images/2.jpg" }
  ]
}
```

### POST /api/cart/items

**リクエストボディ**

```json
{
  "product_id": 1,
  "quantity": 2
}
```

**レスポンス (201 Created)**

```json
{
  "cart_item_id": 42,
  "message": "カートに追加しました"
}
```

## 3. DB アクセス詳細

### テーブル: `product_images`

| カラム | 型 | 説明 |
|--------|----|------|
| id | INT PK | |
| product_id | INT FK | |
| image_url | VARCHAR(500) | |
| sort_order | INT | 表示順 |

### テーブル: `cart_items`

| カラム | 型 | 説明 |
|--------|----|------|
| id | INT PK | |
| session_id | VARCHAR(128) | セッションID |
| product_id | INT FK | |
| quantity | INT | |
| created_at | DATETIME | |
| updated_at | DATETIME | |

### カート追加 UPSERT SQL

```sql
INSERT INTO cart_items (session_id, product_id, quantity, created_at, updated_at)
VALUES (:session_id, :product_id, :quantity, NOW(), NOW())
ON DUPLICATE KEY UPDATE
  quantity = quantity + :quantity,
  updated_at = NOW();
```

---

## 4. 税込価格計算

- 消費税率: 10%（定数管理、将来の税率変更に対応）
- 計算式: `price_with_tax = FLOOR(price * 1.10)`

## 5. エラーハンドリング

| エラー | 対応 |
|--------|------|
| 商品が存在しない / 非公開 | 404ページ表示 |
| 在庫不足 | エラーメッセージ「在庫が不足しています」表示 |
| カート追加失敗（DB） | エラーメッセージ表示、ログ出力 |
