# 機能設計書 – 商品詳細画面

| 項目 | 内容 |
|------|------|
| 画面ID | SCR-002 |
| 画面名 | 商品詳細 |
| 作成日 | 2026-04-08 |
| バージョン | 1.0 |

---

## 1. 概要

特定の商品の詳細情報（画像・説明・価格・在庫）を表示し、カートへの追加を行う画面。

---

## 2. 画面レイアウト

```
┌─────────────────────────────────────────┐
│  ヘッダー（ロゴ / カートアイコン）      │
├──────────────────┬──────────────────────┤
│  商品画像（メイン）│  商品名              │
│                  │  価格（税込表示）     │
│  サムネイル一覧  │  在庫状況            │
│                  │  数量セレクタ        │
│                  │  [カートに追加]      │
│                  │  [お気に入り登録]    │
├──────────────────┴──────────────────────┤
│  商品説明（リッチテキスト）              │
├─────────────────────────────────────────┤
│  関連商品一覧                           │
└─────────────────────────────────────────┘
```

---

## 3. 機能一覧

| No | 機能名 | 説明 |
|----|--------|------|
| F-01 | 商品情報表示 | 商品ID に対応する詳細情報を表示する |
| F-02 | 画像ギャラリー | メイン画像とサムネイルをスライダーで表示する |
| F-03 | 数量選択 | 購入数量を1〜在庫数の範囲で選択する |
| F-04 | カートへ追加 | 指定数量をカートに追加する |
| F-05 | お気に入り登録 | ログイン済みユーザーがお気に入りに登録できる |
| F-06 | 関連商品表示 | 同一カテゴリの商品を最大4件表示する |

---

## 4. 画面遷移

| 操作 | 遷移先 |
|------|--------|
| 「カートに追加」押下 | カート画面（SCR-003） |
| 関連商品クリック | 商品詳細画面（SCR-002）※別商品 |
| パンくずリスト | 商品一覧画面（SCR-001） |

---

## 5. 業務ルール

- 在庫数が0の場合、数量セレクタとカート追加ボタンを非表示にし「在庫なし」表示
- 数量は最大9個まで選択可（在庫数が9未満の場合は在庫数が上限）
- 未ログインユーザーがお気に入りボタンを押した場合、ログイン画面へ遷移
- 非公開商品のURLへ直接アクセスした場合、404ページを表示
