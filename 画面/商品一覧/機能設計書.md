## 1. 概要

ユーザーが販売中の商品を閲覧し、絞り込みや並び替えを行う画面。
商品詳細画面・カート画面へのエントリーポイントとなる。

**対象ユーザー**: ゲスト（未ログイン）・会員（ログイン済み）どちらもアクセス可能。
ただし「カートへ追加」はゲストでも実行可能（セッションベースのゲストカートを利用）。

---

## 2. 画面レイアウト

```
┌─────────────────────────────────────┐
│  ヘッダー（ロゴ / カートアイコン）  │
├─────────────────────────────────────┤
│  カテゴリ絞り込み ／ キーワード検索 │
│  並び替え（新着順 / 価格順 / 人気順）│
├──────────┬──────────┬───────────────┤
│ 商品カード│ 商品カード│ 商品カード    │
│ 画像      │ 画像      │ [在庫なし]    │
│ 商品名    │ 商品名    │ 商品名        │
│ 価格      │ 価格      │ 価格          │
│[カートへ] │[カートへ] │（ボタン非活性）│
├─────────────────────────────────────┤
│  ページネーション                   │
├─────────────────────────────────────┤
│  フッター（共通コンポーネント）     │
└─────────────────────────────────────┘
```

**レスポンシブ対応**: スマートフォン・タブレット対応必須。モバイルでは1カラム表示に切り替える。

**共通部品**: ヘッダー・フッターは共通コンポーネントを使用（別途共通部品仕様書を参照）。

---

## 3. 機能一覧

| No | 機能名 | 説明 |
|----|--------|------|
| F-01 | 商品一覧表示 | 公開中の商品をページネーション付きで表示する |
| F-02 | カテゴリ絞り込み | カテゴリ選択により表示商品を絞り込む（デフォルト: 全カテゴリ） |
| F-03 | キーワード検索 | 商品名・説明文を対象に部分一致（LIKE前後方一致）でキーワード検索する |
| F-04 | 並び替え | 新着順・価格昇順/降順・人気順（累計販売数による）で並び替える |
| F-05 | カートへ追加 | 一覧から直接カートに商品を追加する（在庫あり商品のみ） |
| F-06 | 商品詳細遷移 | 商品カードクリックで商品詳細画面へ遷移する |
| F-07 | ゼロ件表示 | 検索結果が0件の場合「該当する商品が見つかりませんでした」と表示し、絞り込みクリアリンクを提示する |
| F-08 | エラー表示 | API/DBエラー時にエラーメッセージを表示する |

---

## 4. 入力項目

| 項目名 | 種別 | 必須 | 説明 |
|--------|------|------|------|
| キーワード | テキスト | 任意 | 最大100文字。半角英数・全角・記号入力可。禁止文字なし。空白のみは検索対象外 |
| カテゴリ | セレクトボックス | 任意 | マスタより動的生成。初期値「全て」 |
| 並び替え | ラジオボタン | 任意 | デフォルト：新着順 |

---

## 5. 画面遷移

| 操作 | 遷移先 |
|------|--------|
| 商品カードクリック | 商品詳細画面（SCR-002） |
| 「カートへ」ボタン押下 | カート画面（SCR-003）※在庫あり時 |
| カートアイコン押下 | カート画面（SCR-003） |
| システムエラー発生 | エラー画面（SCR-ERR） |

**ブラウザバック時**: 商品詳細・カートからの戻り時、絞り込み条件・ページ位置を保持する（URL パラメータで状態を管理）。

---

## 6. 表示件数

- 1ページあたり 20件表示（固定）
- 最大ページ数: 500ページ（10,000件上限）。上限超過分は表示しない。

---

## 7. 業務ルール

- 在庫数が0の商品は「在庫なし」を表示し、カート追加ボタンを非活性にする
- 公開フラグが「公開」の商品のみ表示する（販売期間外・取扱停止商品は表示しない）
- カテゴリ未選択時は全カテゴリを対象とする
- ゲストも会員も同一の商品一覧を閲覧可能。カート追加はゲストも可能（セッションカート）
- ロール別制御: 管理者専用商品等は対象外。会員・ゲスト間の表示差異なし
- 価格表示: 税込価格を表示する（消費税法に基づく総額表示義務）
- 「人気順」は累計販売数（`sales_count`）の降順。販売数同数の場合は新着順
- 検索結果ゼロ件・全商品非公開・カテゴリに商品なしの場合は F-07 のメッセージを表示
