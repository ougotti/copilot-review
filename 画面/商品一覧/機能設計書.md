# 機能設計書 – 商品一覧画面

| 項目 | 内容 |
|------|------|
| 画面ID | SCR-001 |
| 画面名 | 商品一覧 |
| 作成日 | 2026-04-08 |
| バージョン | 1.0 |

---

## 1. 概要

ユーザーが販売中の商品を閲覧し、絞り込みや並び替えを行う画面。
商品詳細画面・カート画面へのエントリーポイントとなる。

---

## 2. 画面レイアウト

```
┌─────────────────────────────────────┐
│  ヘッダー（ロゴ / カートアイコン）  │
├─────────────────────────────────────┤
│  カテゴリ絞り込み ／ キーワード検索 │
│  並び替え（新着順 / 価格順 / 人気順）│
├──────────┬──────────┬───────────────┤
│ 商品カード│ 商品カード│ 商品カード    │
│ 画像      │ 画像      │ 画像          │
│ 商品名    │ 商品名    │ 商品名        │
│ 価格      │ 価格      │ 価格          │
│[カートへ] │[カートへ] │[カートへ]     │
├─────────────────────────────────────┤
│  ページネーション                   │
└─────────────────────────────────────┘
```

---

## 3. 機能一覧

| No | 機能名 | 説明 |
|----|--------|------|
| F-01 | 商品一覧表示 | 公開中の商品をページネーション付きで表示する |
| F-02 | カテゴリ絞り込み | カテゴリ選択により表示商品を絞り込む |
| F-03 | キーワード検索 | 商品名・説明文を対象にキーワード検索する |
| F-04 | 並び替え | 新着順・価格昇順/降順・人気順で並び替える |
| F-05 | カートへ追加 | 一覧から直接カートに商品を追加する |
| F-06 | 商品詳細遷移 | 商品カードクリックで商品詳細画面へ遷移する |

---

## 4. 入力項目

| 項目名 | 種別 | 必須 | 説明 |
|--------|------|------|------|
| キーワード | テキスト | 任意 | 最大100文字 |
| カテゴリ | セレクトボックス | 任意 | マスタより動的生成 |
| 並び替え | ラジオボタン | 任意 | デフォルト：新着順 |

---

## 5. 画面遷移

| 操作 | 遷移先 |
|------|--------|
| 商品カードクリック | 商品詳細画面（SCR-002） |
| 「カートへ」ボタン押下 | カート画面（SCR-003）※在庫あり時 |
| カートアイコン押下 | カート画面（SCR-003） |

---

## 6. 表示件数

- 1ページあたり 20件表示（固定）

---

## 7. 業務ルール

- 在庫数が0の商品は「在庫なし」を表示し、カート追加ボタンを非活性にする
- 公開フラグが「公開」の商品のみ表示する
- カテゴリ未選択時は全カテゴリを対象とする
