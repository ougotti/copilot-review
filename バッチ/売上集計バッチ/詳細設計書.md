## 3. 集計 SQL

### 日次サマリー集計

```sql
SELECT
  DATE(o.created_at)        AS sales_date,
  SUM(o.total_amount)       AS total_amount,
  COUNT(o.id)               AS order_count
FROM orders o
WHERE o.payment_status = 'paid'
  AND DATE(o.created_at) = :target_date
GROUP BY DATE(o.created_at);
```

### 商品別集計

```sql
SELECT
  DATE(o.created_at)        AS sales_date,
  oi.product_id,
  oi.product_name,
  p.category_id,
  SUM(oi.quantity)          AS quantity_sold,
  SUM(oi.unit_price * oi.quantity) AS total_amount
FROM order_items oi
JOIN orders o ON oi.order_id = o.id
JOIN products p ON oi.product_id = p.id
WHERE o.payment_status = 'paid'
  AND DATE(o.created_at) = :target_date
GROUP BY oi.product_id, oi.product_name, p.category_id;
```

### sales_daily UPSERT

```sql
INSERT INTO sales_daily (sales_date, total_amount, order_count, created_at, updated_at)
VALUES (:sales_date, :total_amount, :order_count, NOW(), NOW())
ON DUPLICATE KEY UPDATE
  total_amount = :total_amount,
  order_count  = :order_count,
  updated_at   = NOW();
```
